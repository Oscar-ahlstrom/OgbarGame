* How do i play this?

Navigate to the repo main page and click the green code button
and clone the repo either through opening with github desktop or by copying the link 

<img width="444" height="428" alt="image" src="https://github.com/user-attachments/assets/e129de8b-2087-42f6-8676-9562cd756f6a" />

if you chose to copy the web URL you need to paste it into github desktop by going into File -> Clone Repository -> URL -> *Paste the URL* -> clone

<img width="973" height="693" alt="image" src="https://github.com/user-attachments/assets/58ba8b85-959e-4281-8f8a-dfbc572da0c5" />

<img width="967" height="683" alt="image" src="https://github.com/user-attachments/assets/fcbcbc53-56d4-4212-8805-9a52151f829f" />
